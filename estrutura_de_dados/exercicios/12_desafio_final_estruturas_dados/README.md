## Objetivo

Aplicar Strings, Listas, Tuplas, Dicionários e Sets em um único programa.

## Enunciado

Crie:

### String

Nome do funcionário.

### Lista

Projetos em que o funcionário participa.

### Tupla

Data de admissão:

```python
(dia, mes, ano)
```

### Dicionário

Informações do funcionário:

```python
{
    "nome": "",
    "idade": "",
    "cargo": ""
}
```

### Set

Tecnologias conhecidas pelo funcionário.

Exiba todas as informações de forma organizada.
