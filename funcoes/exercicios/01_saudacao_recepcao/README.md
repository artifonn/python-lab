## Contexto

Uma empresa está implantando um sistema interno para recepcionar visitantes e colaboradores. Sempre que alguém acessar a aplicação, uma mensagem de boas-vindas deverá ser exibida para tornar a experiência mais amigável e profissional.

## Objetivo

Aprender a criar funções em Python.

## Enunciado

Crie uma função chamada `saudacao()`.

A função deverá exibir a seguinte mensagem:

- Seja bem-vindo ao sistema da empresa.

Após criar a função, execute-a para que a mensagem seja exibida no terminal.
